InvestSimples API

API Financeira moderna com JWT + Entity Framework Core

.NET 9 EF Core SQLite JWT Auth
1

Abra o Swagger UI

A interface visual oficial para explorar e testar todos os endpoints da API:

Abrir Swagger UI

URL direta: http://localhost:5001/swagger

2

Faça Login

No Swagger, clique em POST /api/auth/login → Try it out e use as credenciais:

{
  "email": "teste@investsimples.com",
  "senha": "123456"
}

Copie o token da resposta (string longa iniciando com eyJhbGciOiJ...)

3

Autorize

No topo do Swagger, clique no botão Authorize 🔒, cole o token com prefixo Bearer :

Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...

Clique Authorize → Close. Pronto! Todos os endpoints protegidos funcionarão.

4

Teste os Endpoints

Clique em qualquer endpoint → Try it out → Execute para ver a resposta em tempo real:

GET

/api/cotacoes

Cotações em tempo real: Dólar, Euro, Bitcoin, Ibovespa, CDI, Selic

GET

/api/carteira

Sua carteira: ações, renda fixa, fundos com rentabilidade e risco

POST

/api/simulador

Simula rendimento com juros compostos (não salva no banco)

POST

/api/simulador/salvar

Simula + salva no histórico do usuário autenticado

GET

/api/simulador/historico

Lista todas simulações salvas com data e resultado

5

Exemplo de Simulação

No POST /api/simulador/salvar, envie:

{
  "valorInicial": 1000,
  "anos": 10,
  "idade": 30,
  "tipoInvestimento": "Ações",
  "taxaAnual": 12
}

Retorna: valor final, total investido, rendimento, idade final
