# 📍 Roadmap do InvestSimples App

Este roadmap define as etapas de desenvolvimento do aplicativo InvestSimples, voltado para iniciantes no mundo dos investimentos.

---

## ✅ Fase 1: Estrutura Inicial

- [x] Criar repositório no GitHub
- [x] Definir estrutura de pastas (frontend, backend, docs)
- [x] Criar README.md com visão geral
- [x] Iniciar projeto Expo com React Native
- [x] Configurar backend com Node.js + Express

---

## 🚧 Fase 2: Funcionalidades Básicas

- [ ] Tela de Simulador de Investimento
- [ ] Conexão entre frontend e backend
- [ ] Validação de dados no simulador
- [ ] Exibição de retorno estimado
- [ ] Tela de conteúdo educativo (artigos, vídeos)

---

## 🔐 Fase 3: Usuário e Autenticação

- [ ] Tela de login/cadastro
- [ ] Integração com Firebase Auth
- [ ] Perfil do usuário com histórico de simulações
- [ ] Proteção de rotas privadas

---

## 💳 Fase 4: Pagamentos e Monetização

- [ ] Integração com Mercado Pago ou PagSeguro
- [ ] Tela de planos e assinatura
- [ ] Registro de transações
- [ ] Área premium com conteúdo exclusivo

---

## 🚀 Fase 5: Refinamento e Lançamento

- [ ] Testes em dispositivos reais
- [ ] Ajustes de UI/UX
- [ ] Publicação na Play Store e App Store
- [ ] Divulgação nas redes sociais
- [ ] Coleta de feedback dos usuários

---

## 📌 Extras e Melhorias Futuras
